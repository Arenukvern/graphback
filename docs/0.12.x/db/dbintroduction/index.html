Database

Graphback provides a rich set of tools and APIs that help you get your database working with your GraphQL application in minutes.

Graphback has a CRUD abstraction with a number of out-of-the-box implementations for different datasources (see Data sources).

Graphback integrates with graphql-migrations so that you can migrate your database using your GraphQL schema. graphql-migrations can be used with the graphback db subcommand or programatically from your application code using the migrateDB API. For more see Database Migrations.
